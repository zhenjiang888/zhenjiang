講義内容

The aim of this course is to teach the basics of functional programming, using the modern functional language Haskell. Broadly speaking, a functional programming language is one that emphasizes the application of functions, rather than the execution of commands. Among other features, functional languages offer a compact notation for writing programs, powerful abstraction methods for structuring programs, and a simple mathematical basis that supports reasoning about programs. An important theme of the course is how to improve programs by manipulating them as if they were formulae in algebra. In many examples, we start with an easy-to-understand program that is inefficient, and manipulate it to obtain a more efficient program that may be more complicated.
