計算機言語論（特論）

Principle of Programming Langauges

TEXTBOOK

Ravi Sethi, Programming Languages: Concepts and Constructs, 2nd Ed., Addison-Wesley Publishing Co., Reading, MA, 1996.

Topics to be covered:

Introduction
　 programming language history; programming paradigms; syntax; semantics
　
Imperative Languages
　 control structures; structured programming
　 data representation; data types; type checking
　 procedures; parameter passing; variable scope; storage allocation
　
Object-Oriented Languages
　 modules; classes; type parameters
　 objects; inheritance; polymorphism
　
Functional Languages
　 higher-order functions; recursion
　
Logic Languages
　 declarative specification; operational interpretation
